# agentic-starter-kit Claude Plugin (Template)

This template provides a plugin-ready structure you can copy into a project.

## Included
- `.claude-plugin/plugin.json`
- `skills/` (placeholder)
- `commands/` (placeholder)
- `hooks/` (placeholder)

## Suggested next steps
1. Copy your core skills into `skills/`
2. Add project-safe slash commands in `commands/`
3. Add non-destructive hooks in `hooks/`
4. Register plugin in your Claude Code settings
